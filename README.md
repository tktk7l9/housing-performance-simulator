# 住宅性能シミュレーター

**[▶ 公開サイト](https://housing-performance-simulator.vercel.app)**

施主が住宅の性能仕様（断熱・気密）と設備（太陽光・蓄電池）の選択を、**初期費用と30年ランニングコストの両面**から中立的に意思決定できるツール。

## 特徴

- **数字で意思決定**: 営業トークではなく、計算根拠を開示しながら30年累計コストを比較
- **中立性**: メーカー名・商品名は出さず、機器カテゴリと効率値のみで扱う
- **透明性**: 計算で使用している全ての係数・前提値を結果画面の「計算根拠」から確認可能
- **共有可能**: 結果ページの URL 共有 / PDF 出力に対応

## 開発

```bash
npm install
npm run dev
```

## 計算の前提

`src/lib/housing/data/` に集約：

- `regions.ts` … 1〜8地域の暖房デグリーデー (HDD)・冷房デグリーデー (CDD)
- `solarIrradiance.ts` … 地域別の年間日射量（NEDO 公開値ベース）
- `insulationPresets.ts` … 省エネ基準・ZEH・HEAT20 G1/G2/G3 の UA・C 値
- `equipment.ts` … エアコン COP / エコキュート COP / ガス給湯効率
- `subsidies.ts` … 代表的な国・地方補助金（`lastUpdated` 必須）

## 注意

実際の光熱費は気象・生活パターン・実機効率により大きく変動する。本ツールの試算は意思決定の比較材料であり、実測との一致を保証しない。
